# 共享库文件 (Shared Libraries)

这个目录包含了可以在多个项目中复用的工具库和组件。

## 📁 目录结构

```
shared-libs/
├── tracker/              # 事件追踪器
│   ├── tracker.js        # 完整版本（支持多环境）
│   ├── tracker-simple.js # 简化版本（微信小程序专用）
│   └── README.md         # 使用说明
└── README.md             # 本文件
```

## 🎯 使用说明

### 复制到其他项目
1. 将需要的库文件复制到目标项目的相应目录
2. 按照各库的 README 说明进行配置和使用

### 版本管理
- 建议使用 Git 标签管理版本
- 在复制到其他项目时，记录版本信息

## 📋 库列表

### 1. Tracker - 事件追踪器
- **用途**: 数据埋点和上报
- **适用环境**: 微信小程序、Web、Node.js
- **文件**: `tracker/tracker.js` 或 `tracker/tracker-simple.js`

## 🔄 更新流程

1. 在 `shared-libs` 中开发和测试
2. 测试完成后，复制到需要使用的项目中
3. 记录版本和更新日志

## 📝 注意事项

- 这些文件不会被打包到微信小程序中
- 仅供开发和参考使用
- 建议定期同步更新到使用这些库的项目中
